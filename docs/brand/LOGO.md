# Kalmio Logo

The selected Kalmio logo direction is **3D. Reserved K Refined**.

Rationale:

- Strong app-icon presence through the primary dark navy field.
- Compact route-built `K` monogram that remains legible at small sizes.
- Assistant violet node is present but restrained.
- Charging green endpoint signals EV context without making the mark a generic charger-map icon.
